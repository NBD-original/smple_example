## **Описание проекта**

Программа представляет собой **калькулятор индекса массы тела (Body Mass Index, BMI)**.  
Пользователь вводит свой рост и вес, после чего приложение рассчитывает BMI и выводит интерпретацию результата согласно классификации Всемирной организации здравоохранения (ВОЗ).

Этот проект выбран как простой и понятный пример программного решения, который легко использовать для демонстрации CI/CD, Git-flow и документации.
## **Функциональность**

Программа:
- запрашивает у пользователя вес (в килограммах),
- запрашивает рост (в метрах),
- рассчитывает BMI по формуле:
    ```
    BMI = вес / (рост²)
    ```
    
- выводит человеко-понятное заключение:
    - недостаточная масса,
    - норма,
    - избыточная масса,
    - ожирение.

## **Как запустить**

Убедитесь, что установлен Python 3.
### 1. Клонируйте репозиторий:

```bash
git clone https://github.com/NBD-original/smple_example.git
cd smple_example
```
### 2. Запустите программу:

```bash
python3 bmi_calculator.py
```
### 3. Пример работы:

```
=== Калькулятор индекса массы тела (BMI) ===
Введите ваш вес (кг): 72
Введите ваш рост (в метрах): 1.80

Ваш BMI: 22.22
Интерпретация: Норма
```
